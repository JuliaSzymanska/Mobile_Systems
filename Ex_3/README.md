# Ex_3 - Paint Application

## About <a name = "about"></a>

The task is to create a simple Paint Application.

## Available features
* Normal brush
* Blur brush
* Eraser
* Change paint's colour
* Change paint's size
* Change canva's colour
* Clear canva
* Upload image
* Save image
